### 介绍
>
> 一款nice的基于 vue 的博客模板。
> 在一些布局样式上借鉴了[Akina For Typecho](https://zhebk.cn/Web/Akina.html)的风格，也做了一些自己的改动。  
> 打算用做搭建自己的博客使用，也开源给大家希望能喜欢，欢迎star。
>

```

```
